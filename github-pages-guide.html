什么是 GitHub？

一句话解释 + 通俗比喻

📌 一句话解释

GitHub 是全球最大的代码托管平台，就像一个"程序员的网盘" + "社交平台"的结合体。

🏠 通俗比喻

概念 类比
GitHub 程序员的"百度网盘" + "朋友圈"
仓库 (Repository) 一个项目文件夹
代码提交 (Commit) 保存文件的历史版本
分支 (Branch) 项目的不同版本线
Fork 复制别人的项目到自己账号
Pull Request 向原作者提交修改建议

⚡ 核心功能

  • 代码托管 - 存储代码文件，记录每次修改历史，随时回滚到任意版本
  • 协作开发 - 多人同时编辑一个项目，查看谁改了什么，讨论和审查代码
  • 开源社区 - 全球开发者分享代码，免费学习优秀项目，参与知名开源软件
  • GitHub Pages - 免费静态网站托管，将 HTML/CSS/JS 文件变成可访问的网站，支持自定义域名
🎯

GitHub Pages 部署步骤

手把手教你，5分钟搞定

1

注册 GitHub 账号

  • 访问 https://github.com/signup
  • 使用邮箱注册（支持国内邮箱：QQ邮箱、163邮箱等）
  • 设置用户名和密码
  • 完成邮箱验证
2

创建仓库

  • 登录后点击右上角 + → New repository
  • 仓库名称填写：qilin-castle-chronicle（或你想要的名称）
  • 选择 Public（公开）
  • 勾选 Add a README file
  • 点击 Create repository
3

上传 HTML 文件

  • 进入仓库页面，点击 Add file → Upload files
  • 将你的 index.html 文件拖拽上传
  • 点击 Commit changes
4

启用 GitHub Pages

  • 点击仓库顶部 Settings 标签
  • 左侧菜单选择 Pages
  • Source 选择 Deploy from a branch
  • Branch 选择 main / /(root)
  • 点击 Save
5

获取你的永久链接

  • 等待 1-2 分钟
  • 访问 https://你的用户名.github.io/仓库名/
  • 🎉 完成！你的网站已经上线！
🔑

获取 Personal Access Token

让自动化部署成为可能

🤔 什么是 Token？

Token 相当于一把"钥匙"，让别人（或自动化工具）可以代替你操作 GitHub 仓库。比如让别人帮你上传文件、部署网站。

1

进入 Token 设置页面

登录 GitHub → 点击右上角头像 → Settings → 左侧最下方 Developer settings → Personal access tokens → Tokens (classic)

2

生成新 Token

  • 点击 Generate new token (classic)
  • 填写 Note（备注，如："部署网站"）
  • Expiration 选择过期时间（建议选 No expiration 永久，或自定义时间）
  • 勾选权限：repo（完整控制仓库）
  • 点击页面底部 Generate token
3

保存 Token

  • 立即复制生成的 token（只显示一次！）
  • 格式类似：ghp_xxxxxxxxxxxxxxxxxxxx
  • 保存到安全的地方，不要泄露给他人
⚠️ 安全提醒 Token 相当于你的账号密码，谁拿到都能操作你的仓库！
使用完毕后，建议立即到 https://github.com/settings/tokens 删除。

GitHub 问题大全

小白最常问的问题，都在这里

💰 GitHub 需要花钱吗？

功能 价格 说明
基本功能 🆓 完全免费 托管代码、创建仓库、GitHub Pages
GitHub Pages 🆓 完全免费 无限流量，我们用的就是它
私人仓库 🆓 每月 500MB 公开仓库无限
付费功能 💰 可选 普通用户完全不需要

结论：咱们用的功能全部免费，不需要花一分钱！

💾 GitHub 能用多大空间？

类型 容量 说明
单个仓库 推荐 ≤ 1GB 软限制，超了会有警告
单个文件 ≤ 100MB 超大文件需要用 Git LFS
GitHub Pages 网站 🆓 1GB 普通网页完全够用

结论：做网页、存文档、存代码都绰绰有余！

🔑 Token 以后还能用吗？

Token 删除后不影响已部署的网站！网站依然正常访问。

以后需要更新网页时，有两个选择：

  • 方案一：临时创建新 Token 给自动化工具，部署完成后立即删除（最安全）
  • 方案二：自己登录 GitHub 手动上传文件（不需要 Token）

🌐 GitHub 是否需要外网？

地区 访问情况
国外 直接访问，速度快
中国大陆 需要科学上网工具（VPN/代理）

如果无法访问 GitHub，可以考虑国内替代方案：

  • Gitee（码云） - gitee.com，国内访问快，免费 Pages
  • Coding - coding.net，腾讯旗下，国内稳定

📊 GitHub 能申请多少 Token？

类型 数量限制
Personal Access Token 想申请多少申请多少
每个 Token 的有效期 你自己设置（1天 ~ 永久）

其实你不需要反复申请 Token！

  • 一个 Token 可以管理所有仓库
  • Token 删除后随时可以新建
  • 只需要在需要自动化部署时临时创建

⏰ GitHub 账号能用多久？

永久！

一次注册，终身使用，不需要续费，不需要定期登录验证。
